#Bash lesson 25From over the wire challenge BanditA daemon is listening on port 30002 and will give you the password for bandit25 if given the password for bandit24 and a secret numeric 4-digit pin code. There is no way to retrieve the pin code except by going through all of the 10000 combinations, called brute-forcing.In terminal:$  mktemp -d$ cd # to directory created$ touch script.sh$ chmod +x !$$ nano script.shOn nano #!/bin/bashFor I in {0000..9999}; do	Echo Òthis lvl pass $iÓdone$ ./script.sh > dictionary.txt$ cat dictionary.txt | nc localhost 30002 | grep -v "Wrong"Password will appear.
