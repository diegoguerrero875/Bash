Tip: if your terminal is messed up, try the “reset” command.In terminal: $ ls        #to list all items de current directoryinhere #shown$ cd inhere/ # to move to the inhere directory$ file ./-* #will show us that file07 is the only ASCII text the other docs are data$ cat ./-file07Password will appear.
