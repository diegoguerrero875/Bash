#Bash lesson 14From over the wire challenge BanditThe password for the next level is stored in /etc/bandit_pass/bandit14 and can only be read by user bandit14. For this level, you don’t get the next password, but you get a private SSH key that can be used to log into the next level. Note: localhost is a hostname that refers to the machine you are working onIn terminal:$ ssh -i sshkey.private bandit14@localhostYou know will be in the next level if you want to have the password for this level you need to do$ cat /etc/bandit_pass/bandit14
