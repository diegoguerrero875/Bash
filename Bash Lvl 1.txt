#Bash lesson 1From over the wire challenge BanditThe password for the next level is stored in a file called readme located in the home directory. Use this password to log into bandit1 using SSH. Whenever you find a password for a level, use SSH (on port 2220) to log into that level and continue the game.In terminal: $ ls        #to list all items de current directory-readme is shown$ cat readme        #to open or read filePassword will appear. With this password go on to lvl 2, first exit the current location with:$ exitThen type again the SSH command but know the user is bandit1, change this number every time you finish a level$ ssh bandit1@bandit.labs.overthewire.org -p 2220Insert the password obtained from this exercise :)
